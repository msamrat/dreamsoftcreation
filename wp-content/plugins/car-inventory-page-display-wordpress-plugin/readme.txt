Inserts a set of form fields into the add post panel to create a car dealer inventory website.

== Description ==

You can display:

* Vehicle information
* Year
* Price
* Features
* Color
* Transmission
* And other related information to the vehicle


Version 1.0
First version of the plugin.  Requires the get custom field plugin.

== Installation ==

1. Upload `cardealer_inventory.php`  to the `/wp-content/plugins/` directory.
2. Activate the plugin via the Plugins menu.
3. Make sure to install and activate The Get Custom Fields plugin for this to work.
4. Insert `[cardealer]' into your post content.
5. Add the new information to the posts.
